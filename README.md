# Basavaraj Ningasani — Portfolio 🚀

A highly interactive, premium personal portfolio built with a mobile-first design philosophy. Inspired by modern "Mindrift" aesthetics, this portfolio features dynamic Framer Motion animations, horizontal snap-scrolling, edge-to-edge typography, and interactive parallax graphics.

## ✨ Key Features

🎨 **Premium Aesthetic**: Heavy typography, high-contrast layouts, glassmorphic headers, and clean background styling.
📱 **Mobile-First UX**: Implements `85vw` horizontal snap-scrolling carousels for easy navigation without long vertical scrolling on mobile devices.
✨ **Micro-Interactions**: Dynamic 3D tilt effects on project cards, interactive parallax background SVGs, and highly-responsive Call-To-Action buttons.
📊 **Dynamic Architecture**: All content is mapped cleanly from modular TS components.

## 📂 Sections
*   **Hero**: High-impact introduction with interactive photo framing and floating parallax SVGs.
*   **Skills**: Clean categorizations featuring a horizontal scroll implementation.
*   **Tech Stack Ticker**: Smooth infinite-scrolling marquees carrying authentic, perfectly-colored original brand logos for all tools.
*   **Projects & Hackathons**: 3D Tilt-effect cards showcasing high-quality implementations with GitHub integration.
*   **Open Source**: Dedicated showcase of merged Pull Requests into large repositories (PyTorch/Electron/Supabase).
*   **Contact**: Fully functional FormSpree email form wrapped into a beautiful UI.

## 🚀 Running Locally

First, install the necessary packages:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## ☁️ Deployment

This project is fully optimized and ready to be deployed on **Vercel** with zero-configuration required. Simply link the GitHub repository and execute deployment.
